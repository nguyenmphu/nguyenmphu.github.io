# Cloud Acquisition
## Fundamentals
### Cloud Adoption Parts
- **Procurement**: When you buy cloud technologies, you are paying for the access to standardized compute, storage, and other IT services.
You are only paying the resources you use.
- **Legal**: Cloud terms and conditions are designed to reflect how cloud services model functions.
- **Security**: CSP is certified for secific third-party accreditation.
- **Governance**: shared responsibility.
- **Finance**: pay-as-you-go utility model & market pricing.
- **Compliance**: work inside certain geographic requirements when procuring cloud.
