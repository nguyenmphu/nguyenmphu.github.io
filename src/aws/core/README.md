## AWS platform architecture
AWS maintains datacenters around the world. They exist within AWS region. You can reduce your own service's network latency by choosing the region that is closest.
> Note that accessing and authenticating to the two Chinese regions requires unique protocols.

Endpoint addresses of AWS resource is formed `<service>.<region>.amazonaws.com`. For example: `ec2.ap-east-1.amazonaws.com`.
