# Container
Containers provide you with a standard way to package your application's code and dependencies into a single object.

AWS container services:

- `ECS (Elastic Container Service)`: highly scalable, high-performance container management system, support Docker.
- `EKS (Elastic Kubernetes Service)`: fully managed service that you can use to run Kubernetes on AWS.`
- `Fargate`: serverless compute engine for containers.
