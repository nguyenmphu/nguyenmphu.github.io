## Prerequisites
- An account Github
- A text editor
- Install hugo
## Steps
1. Create a new github repository.
2. Go to settings, in `Pages` > `Build and Deployment`, select `source` as `Github Action`. You can choose from workflow templates or create your own.
3. Commit the config.
4. In the local machine, pull the new code.
5. Start a new hugo project: `hugo new site . --force`
